----------
Tutorials
----------

This tutorial covers the main features of ``nsbtools``, including geometric eigenmode calculation, 
 decomposition and reconstruction of brain maps, neural field theory simulation, and 
 visualization of brain maps. For further details, please refer to the API documentation. If you
 encounter any issues, please open an issue on the `GitHub repository 
 <https://github.com/NSBLab/nsbtools/issues>`_.
